Parte 2 del rompecabezas. Step over

Para esta página, abre la pestaña Fuentes en las DevTools y utiliza el comando "Step over" botón para encontrar la contraseña secreta.

Consejos y trucos

  1. Para abrir DevTools, puedes hacer clic derecho en la página y elegir la opción "Inspeccionar", o utilizar el atajo de teclado.

    • Mac: Cmd + Option + i
    • Windows/Linux: Ctrl + Shift + i

    Luego actualiza la página con el depurador abierto.

  2. Si el depurador se detiene sin mostrarte el código correcto, haz clic en la primera entrada en la pila de llamadas para saltar adonde realmente se detuvo. Esto suele ocurrir la primera vez que nos detenemos en una determinada página.

  3. Si se detiene en un código que obviamente no forma parte del rompecabezas, probablemente se deba a las extensiones de Chrome. Solo tienes que pulsar play o abrir la página en una ventana de incógnito.
